Layout

Layout med grid

CSS grid layout är en ny css layout modell. Generellt är en sida uppdelad i flera huvudområden, och förhållandet mellan dessa områdens storlek, position och hierarki definieras. Det är för närvarande den enda tvådimensionella css-layouten.

Grid-template-columns, grid-template-rows, grid-template-areas

Egenskapen grid-template-columns anger speciell värdet som man vill ha på olika kolumner i en grid layout. Egenskapen grid-template-rows i CSS används för att ställa in antalet och höjden på rader i grid. Värdena för grid-template-rows är åtskilda av mellanslag, där varje värde representerar höjden på en rad. Grid-template-areas definiera och anger områdena i en CSS grid layouten. Ett Grid-template-areas gör en visuell representation av grid med både kolumner och rader.

Vad är en media query och vad används den till?

Media query är en viktig del av responsiv webbdesign eftersom man kan definiera olika stilar för olika medietyper och skapa olika layouter för rikare gränssnitt. Till exempel kan den realisera rimlig visning av samma innehåll på olika gränssnitt som datorer, mobiltelefoner etc.
